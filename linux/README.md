# Linux

Not yet configured. Linux setup is similar to mac/ with these differences:

- Use `notify-send` instead of `osascript` for notifications
- Shell is likely bash (use bashrc, `read -rsn1`)
- `date` uses GNU syntax (same as Windows/MSYS2)
- Install via: `sudo apt install tmux fzf` (Debian/Ubuntu) or `sudo pacman -S tmux fzf` (Arch)
